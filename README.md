## 스터디 방법 
### 매일
1. 정해진 문제를 푼다
2. 다른 스터디원의 어제 코드에 주석(////)을 달아 피드백 해준다.
3. 그저께 코드에 달린 피드백을 반영한다.
4. 내일 풀 문제를 정한다.

### 규칙
* 소스파일 이름은 boj0000_hgd 으로 통일한다.
