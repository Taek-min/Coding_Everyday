## 📋 스터디 방법
### 📌 매일
1. 정해진 한 문제를 푼다.
2. 다른 스터디원의 어제 코드에 주석(////)을 달아 피드백 해준다.
3. 그저께 내 코드에 달린 피드백을 반영한다.
4. 내일 풀 문제를 정한다.

### 📌 규칙
* 2~3시간 이상 충분히 고민했는데도 풀리지 않는다면 반례나 답안 소스를 참고한다.
* 아래의 경우 스터디에서 제외된다.
  * 2일 연속으로 문제 풀이가 밀렸을 때
  * 밀리는 경우가 너무 잦을 때
